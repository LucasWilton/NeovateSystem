Sobre o projeto

Transporte inteligente para motoristas reais

A organização manual de rotas gera atrasos, desperdício de combustível e desgaste operacional. O Trajetto resolve esse problema com tecnologia acessível.

Desenvolvido como projeto acadêmico na disciplina de Fábrica de Software I da UNIFACOL, por uma equipe de 4 pessoas com foco em Sistemas de Informação.

  • Cadastro de passageiros e pontos de embarque
  • Geração automática de rotas otimizadas via Google Maps API
  • Visualização interativa dos trajetos em mapa
  • Histórico e edição dinâmica de rotas

Funcionalidades

O que o Trajetto oferece

Um sistema web focado em resolver problemas reais do transporte por vans escolares e fretamento.

Cadastro de usuários

Motoristas e empresas se cadastram com perfis distintos, cada um com funcionalidades específicas ao seu contexto.

Pontos de embarque

Cadastre passageiros e seus respectivos endereços de embarque, integrados com geolocalização.

Otimização de rotas

O sistema calcula automaticamente a melhor sequência de paradas via API do Google Maps, minimizando distância e tempo.

Histórico de trajetos

Acesse trajetos anteriores, compare desempenho e reutilize rotas de dias anteriores com facilidade.

Dashboard analítico

Visualize em tempo real rotas ativas, passageiros a bordo, economias de combustível e desempenho semanal.

Interface intuitiva

Design focado na usabilidade do motorista, com fluxos simples e acessíveis mesmo para usuários com pouca familiaridade com tecnologia.

React Node.js PostgreSQL Google Maps API MVC Architecture Scrum REST API
